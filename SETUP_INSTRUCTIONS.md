# Инструкция по настройке проекта

## Шаг 1: Создание сцены сборки

1. В Unity создайте новую сцену и сохраните её как `DroneAssembly`

2. **Создайте структуру сборки:**
   - Используйте меню `Tools > Drone Assembly Setup Helper`
   - Нажмите "Создать базовую структуру сборки"
   - Это автоматически создаст раму, слоты для деталей и менеджер

3. **Создайте детали квадрокоптера:**
   - Для каждой детали создайте GameObject (можно использовать примитивы Unity)
   - Добавьте компонент `DronePart` на каждую деталь
   - Установите правильный `Part Type` для каждой детали
   - Разместите детали в начальных позициях (например, на панели слева)

   **Необходимые детали:**
   - Frame (Рама) - 1 шт
   - Motor1, Motor2, Motor3, Motor4 (Моторы) - 4 шт
   - Propeller1, Propeller2, Propeller3, Propeller4 (Пропеллеры) - 4 шт
   - Battery (Батарея) - 1 шт
   - FlightController (Контроллер полета) - 1 шт

4. **Настройте UI:**
   - Создайте Canvas (UI > Canvas)
   - Добавьте кнопку "Запустить полет" (UI > Button)
   - Добавьте текст для отображения прогресса (UI > Text)
   - Добавьте компонент `AssemblyUI` на Canvas
   - Назначьте ссылки на кнопку и текст в инспекторе

5. **Назначьте слоты в менеджере:**
   - Выберите объект AssemblyManager
   - Используйте `Tools > Drone Assembly Setup Helper > Найти все слоты`
   - Или вручную перетащите все слоты в список `All Slots`

## Шаг 2: Создание сцены симулятора полета

1. Создайте новую сцену и сохраните её как `FlightSimulator`

2. **Создайте квадрокоптер:**
   - Создайте пустой GameObject "Drone"
   - Добавьте Rigidbody:
     - Mass: 1
     - Use Gravity: ✓
     - Drag: 5
     - Angular Drag: 5
   - Добавьте компонент `DroneController`
   
3. **Создайте модель квадрокоптера:**
   - Создайте раму (Cube, масштаб 2x0.2x2)
   - Создайте 4 мотора (Cylinder или Sphere) и разместите их по углам рамы
   - Создайте 4 пропеллера (Cylinder, плоский) и разместите над моторами
   - Сделайте пропеллеры дочерними объектами дрона
   - Назначьте пропеллеры в `DroneController` (Propeller1-4)

4. **Настройте камеру:**
   - Выберите Main Camera
   - Добавьте компонент `CameraFollow`
   - Установите Drone как Target
   - Настройте Offset (например, 0, 5, -10)

5. **Создайте окружение:**
   - Создайте Plane для земли (GameObject > 3D Object > Plane)
   - Добавьте освещение (Directional Light)
   - Опционально: добавьте препятствия или цели

6. **Настройте UI:**
   - Создайте Canvas
   - Добавьте текстовые элементы:
     - Throttle Text (для отображения газа)
     - Altitude Text (для высоты)
     - Speed Text (для скорости)
   - Добавьте компонент `FlightUI` на Canvas
   - Назначьте все ссылки в инспекторе

7. **Добавьте менеджер симулятора:**
   - Создайте пустой GameObject "FlightSimulatorManager"
   - Добавьте компонент `FlightSimulatorManager`
   - Создайте пустой GameObject "SpawnPoint" и разместите его над землей
   - Назначьте SpawnPoint в менеджере

## Шаг 3: Настройка Build Settings

1. Откройте `File > Build Settings`
2. Добавьте обе сцены:
   - `DroneAssembly` (должна быть первой)
   - `FlightSimulator`
3. Убедитесь, что `DroneAssembly` отмечена как стартовая сцена

## Шаг 4: Тестирование

1. Запустите игру
2. В сцене сборки перетащите детали на правильные места
3. Когда все детали установлены, кнопка "Запустить полет" станет активной
4. Нажмите кнопку для перехода в симулятор
5. Используйте клавиши управления для полета

## Советы по настройке

- **Размеры деталей:** Используйте реалистичные размеры (рама ~2x2 единицы, моторы ~0.3 единицы)
- **Позиции слотов:** Размещайте слоты точно там, где должны быть детали
- **Триггеры:** Убедитесь, что у всех слотов есть триггер-коллайдеры
- **Физика:** Настройте параметры Rigidbody для реалистичного полета
- **Камера:** Настройте offset и smooth speed для комфортного обзора

## Устранение проблем

- **Детали не устанавливаются:** Проверьте, что у деталей и слотов есть коллайдеры
- **Дрон не взлетает:** Увеличьте `Motor Power` в `DroneController`
- **Дрон нестабилен:** Настройте `Stability` и `Drag` параметры
- **Камера не следует:** Проверьте, что Target назначен в `CameraFollow`
